# Getting Started with Nest.js

## Prerequisites

- [pnpm](https://pnpm.io/9.x/installation#on-windows)
- Node.js ([use pnpm](https://pnpm.io/9.x/cli/env) to install or install via [node.org](https://nodejs.org/en))
- [learn how to setup](https://github.com/kelcho-spense/node-env-setup)

## Installation

```bash
pnpm i -g @nestjs/cli
nest new project-name
```
